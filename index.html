  • Agent 應用：體驗由多個專業 AI Agent 協同合作，自動化完成簡報製作的進階流程。

簡報的五大核心任務

任務 核心目標
1. 定義目標 (Define)確認簡報目的、聽眾是誰，決定溝通的終點。
2. 規劃結構 (Structure)建立清晰的敘事框架，引導聽眾的思考路徑。
3. 產出內容 (Create)將資料轉化為精煉的文字與關鍵訊息。
4. 視覺設計 (Design)透過排版、配色與圖像，強化訊息的傳達力。
5. 演練與修正 (Rehearse)模擬報告情境，確保流程順暢並優化表達方式。

工具名稱 強項 推薦場景

黃金結構與視覺原則

三大黃金簡報結構

對比式

透過「現況 vs. 未來」的強烈對比，凸顯方案價值。

問題解法式

定義問題 → 分析原因 → 提出解方。邏輯清晰，適用性廣。

敘事式

以故事或案例貫穿全場，引發聽眾共鳴。

三大關鍵視覺原則

一致性

重複使用相同字體、配色與版型，建立專業感。

對比

利用顏色、大小、粗細的差異，凸顯關鍵資訊。

留白

適當留白能讓畫面更清爽，內容更容易閱讀。

Prompt 優化心法

                         
除了上述設定，你還可以在 User Prompt 中手動加入這些元素，讓結果更完美：

                         

                             
核心訊息 (Core Message)
如果聽眾只能記住一件事，那會是什麼？明確地告訴 AI。

                              
提供範例 (Examples)
提供一個好的或壞的範例讓 AI 參考。例如：請參考這段文字的風格，但避免使用術語。

                              
呼籲行動 (Call to Action)
明確指出你希望聽眾採取的下一步是什麼。
